# Reverse Stack using Recursion

## Approach
- Pop top element
- Recursively reverse remaining stack
- Insert element at bottom

## Complexity
- Time: O(n^2)
- Space: O(n) recursion stack

## Key Learning
Recursion can simulate stack operations without extra DS.

# Delete Middle Element of Stack

## Approach
- Use recursion
- Remove k-th element from top (middle)

## Complexity
- Time: O(n)
- Space: O(n)

## Key Learning
Recursion helps access internal elements of stack.
